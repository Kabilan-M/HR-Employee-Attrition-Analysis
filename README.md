# HR Employee Attrition Analysis

This project analyzes employee attrition data using Python, Pandas and Matplotlib in Google Colab.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Charts Created
1. Employee Attrition Count
2. Attrition Donut Chart
3. Department-wise Attrition
4. Gender-wise Attrition
5. OverTime-wise Attrition
6. Job Satisfaction-wise Attrition
7. Work-Life Balance-wise Attrition
8. Average Monthly Income by Attrition

## Conclusion
The analysis shows that overtime, low job satisfaction, poor work-life balance and income level can affect employee attrition.
